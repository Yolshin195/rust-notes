## 🛠 Моё решение

```rust
let mut head = Some(&first);

loop {
    if head.is_none() {
        break;
    }

    let node = head.unwrap();
    println!("{}", node.value);

    if let Some(box_node) = &node.next {
        head = Some(box_node);
    } else {
        head = None;
    }
}
```

Работает.

Но внутри ощущение:

> "блин… как-то громоздко"


## ✨ Rust-style решение

Пошёл копать дальше.

И нашёл вот это:

```rust
let mut head = Some(first.as_ref());

while let Some(node) = head {
    println!("{}", node.value);
    head = node.next.as_ref().map(|n| n.as_ref());
}
```

И тут уже прям кайф.

Коротко. Чисто. Понятно.
